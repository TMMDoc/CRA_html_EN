The control system shall provide the capability to continuously monitor all security mechanism performance using commonly accepted security industry practices and recommendations to detect, characterize and report security breaches in a timely manner.

This SR requires that the system continuously monitor security events, network activity, and system status to detect potential security incidents in real time. Requirement Enhancements (REs) under SR 6.2:

SR 6.2 RE 1 – Automated analysis and correlation of security events.

SR 6.2 RE 2 – Alerting on detected security events.

SR 6.2 RE 3 – Real-time monitoring with minimal delay.
