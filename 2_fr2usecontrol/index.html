FR 2 – Use control

  • Role-Based Access Control (RBAC): Implement RBAC to restrict access to system resources based on the user’s role within the organization.

  • Least Privilege Principle: Ensure users and systems have only the minimum level of access necessary to perform their duties.

    • Praxis has a role-based access control policy for each user

    • Login needs optionally enforced with each startup
